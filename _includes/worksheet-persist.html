{%- comment -%} Saves a page's worksheet fields to the browser's localStorage so a returning visitor keeps their answers. Everything stays on the device; nothing is sent anywhere. Fields are keyed by id or name and namespaced by page path. Include this BEFORE the page's own script, then call WSPersist(opts) from inside that script after any dynamically-built fields exist. opts.onClear is called after a Clear (wire a #ws-clear button). Restored values are read before the page's final calc() runs, so calculators compute from saved data. {%- endcomment -%}
