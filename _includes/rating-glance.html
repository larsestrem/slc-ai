We don't display a numeric rating, star, flag, or finding anywhere on the site (see /guides/how-we-list-communities/ for why) - this gives a visitor an immediate, clearly-labeled set of links to where the real records, reviews, and ratings live. The official state inspections-and-violations record leads. Two profile shapes: - Single-license facility: uses page.records_url / page.cms_ccn / page.licensing. - Multi-license campus: page.state_records is an array of {label, url, cms_ccn?}, one entry per licensed building, rendered under the state agency heading. page.campus_note (optional HTML) adds a muted cross-operator note. {%- endcomment -%}
